In previous version (0.0.1) the averaged values of quantities such as ALBED, QBAK, QEXT, etc. were not coming correctly when a frequesncy distribution is used. This is solved in the version (0.0.2).
In present version (0.0.3) we can add scattering angle 'theta' as an argument - either as a single value or in a range e.g. [10,45].
