API Reference

Detailed listings of the classes, calculated attributes, logged outputs, and interactive help utilities provided by mieshah.

1. miescatter Attributes

These attributes are populated on the miescatter object after initiating a calculation via ms.miescatter(...).

Attribute Type Description
X float Size parameter ($x = 2 \pi a / \lambda$). In polydisperse mode, this represents the average size parameter of the size distribution.
r_eff float Effective radius of the particle size distribution. In monodisperse mode, this equals the input particle radius.
QEXT float Extinction efficiency ($Q_{\text{ext}}$).
QSCA float Scattering efficiency ($Q_{\text{sca}}$).
QABS float Absorption efficiency ($Q_{\text{abs}} = Q_{\text{ext}} - Q_{\text{sca}}$).
ALBED float Single-scattering albedo ($\omega = Q_{\text{sca}} / Q_{\text{ext}}$). For polydisperse mode, computed physically as $\langle Q_{\text{sca}} \rangle / \langle Q_{\text{ext}} \rangle$.
ASYM float Asymmetry parameter ($g = \langle \cos\theta \rangle$).
QPR float Radiation pressure efficiency ($Q_{\text{pr}} = Q_{\text{ext}} - g \cdot Q_{\text{sca}}$).
QBAK float Backscattering efficiency ($Q_{\text{bak}}$).
RHO float Phase shift parameter ($\rho = 2x(m-1)$).
SCA float Integrated scattering cross-section value.
theta list List of scattering angles in degrees for which angular metrics were calculated.
I_perp list Scattering intensity perpendicular to the scattering plane, $I_{\perp}(\theta)$, corresponding to each angle in theta.
I_parl list Scattering intensity parallel to the scattering plane, $I_{\parallel}(\theta)$, corresponding to each angle in theta.
Polar list Degree of linear polarization, $P(\theta) = (I_{\perp} - I_{\parallel}) / (I_{\perp} + I_{\parallel})$, corresponding to each angle in theta.
p_theta list Scattering phase function values corresponding to each angle in theta.

2. spectrum Attributes

These attributes are populated on the spectrum object after initialization. Each attribute contains a list representing values computed at each individual wavelength step in the spectrum.

Attribute Type Description
wl list Scaled wavelengths of the spectrum in microns.
X list Size parameters corresponding to each wavelength.
r_eff list Effective radius of the particle size distribution corresponding to each wavelength.
QEXT list Extinction efficiencies corresponding to each wavelength.
QSCA list Scattering efficiencies corresponding to each wavelength.
QABS list Absorption efficiencies corresponding to each wavelength.
ALBED list Single-scattering albedo values corresponding to each wavelength.
ASYM list Asymmetry parameters corresponding to each wavelength.
QPR list Radiation pressure efficiencies corresponding to each wavelength.
QBAK list Backscattering efficiencies corresponding to each wavelength.
SCA list Integrated scattering values corresponding to each wavelength.
I_perp list Scattering intensities perpendicular to the scattering plane at the configured single theta angle.
I_parl list Scattering intensities parallel to the scattering plane at the configured single theta angle.
Polar list Degrees of linear polarization at the configured single theta angle.
p_theta list Scattering phase function values at the configured single theta angle.

3. Logged CSV Files Layout

When running calculations, CSV logs are exported for post-processing and logging:

  • Single Wavelength logs (mie1.csv and mie2.csv):
    • mie1.csv: Logs bulk averaged efficiency factors and parameters at the reference angle.
    • mie2.csv: Logs angular values (theta, I_perp, I_parl, Polar, p_theta) for all calculated scattering angles.
  • Spectrum logs (Spec_mie1.csv and Spec_mie2.csv):
    • Spec_mie1.csv: Logs spectral progress of bulk parameters (X, WL, QSCA, QEXT, QABS, ALBED, ASYM, QPR, QBAK, SCA, r_eff).
    • Spec_mie2.csv: Logs spectral progress at the chosen theta (wl, theta, I_perp, I_parl, Polar, p_theta).

4. Interactive Help System

Access quick reference help from your active Python terminal session using the built-in help features:

Python Console 
import mieshah as ms

# Show all help documentation
ms.help()

# Or show help for specific components and parameter meanings
ms.Help.show_miescatter()
ms.Help.show_specfile()
ms.Help.show_spectrum()
