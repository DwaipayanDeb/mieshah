Citation & License

Information on citing the mieshah software in journals, research publications, or packages, along with licensing details.

Citing the Software

If you use mieshah in your research, academic paper, or projects, please cite it using the Zenodo software citation format below.

Text Citation

Standard APA Style

Deb, D. (2026). mieshah - A python Mie theory calculator (Version 1.1.1) [Software]. Zenodo. https://doi.org/10.5281/zenodo.15380219

BibTeX Entry

For users writing publications in LaTeX, use the following BibTeX entry:

mieshah.bib 
@software{deb_2026_mieshah,
  author       = {Deb, Dwaipayan},
  title        = {mieshah - A python Mie theory calculator},
  month        = jun,
  year         = 2026,
  publisher    = {Zenodo},
  version      = {1.1.1},
  doi          = {10.5281/zenodo.15380219},
  url          = {https://doi.org/10.5281/zenodo.15380219}
}
