mieshah v1.1.1

A powerful, high-performance Python package for calculating light scattering properties and parameters of spherical particles using Mie theory which is specially designed for astronomical applications. It supports both monodisperse (single particle size) and polydisperse (particle size distributions) systems.

Theoretical Foundation

Developed from a Mie theory foundation originally published in:
Ghanshyam A. Shah, "Numerical Methods for Mie Theory of Scattering by a Sphere", Kodaikanal Obs. Bull. Soc. (1977) 2, 42-63.
This package has been modernized with state-of-the-art numerical calculation methods from Bohren and Huffman (1998) to ensure high performance, numerical stability, and precision.

Developer: Dwaipayan Deb (2026)
Zenodo DOI: 10.5281/zenodo.15380219

Key Features

Validated Results

Yields results numerically identical to established codes (e.g., PyMieScatt, miepython) using Wiscombe's recurrence criteria.

Symbolic Distributions

Specify any size distribution as an arbitrary symbolic string (e.g. f="x**-2" or f="r-**2") and let sympy handle analytical parsing.

Physical Averages

Implements correct physical weighting formulas for size distributions (e.g. ratio-of-averages for albedo) instead of arithmetic averaging.

Flexible Angular Windows

Restrict calculations to specific scattering angle ranges (e.g., theta=[10, 45]) or custom spacings to optimize computational speed.

Spectral Batch Calculations

Easily load refractive index files (wavelength or wavenumber mode) and compute spectra for polydisperse/monodisperse samples in a single step.

Interactive Help

Built-in interactive console help. Simply run help() or call Help.show_all() from your Python session to read guides.

Installation

Install mieshah directly via pip:

Terminal 
pip install mieshah

Dependencies

The package builds on top of standard scientific and numerical computing libraries:

  • numpy - High-performance numerical arrays and grid computations
  • sympy - Symbolic math evaluation for analytical parsing of particle distributions
  • dimpy - Dimension management and conversions
  • tablefile - Reading structured columns and tabular formatting
Quick Tip

Pip automatically resolves and installs all these dependencies during the installation process.
